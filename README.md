# CS3338-Final-Project
2D Sprite-Based Platformer (CS3338 Final Project)
This repository contains the source code and documentation for our team's 2D platformer adventure game. The project focuses on modular architecture, sprite-based physics, and robust quality assurance tracking.

Project Management & QA
We used Jira for task management and TestRail for milestone verification.

Jira Board: View Project Backlog

Testing Documentation: Detailed milestone reports for Snapshots 2, 3, and 4 are located in the /TestRail_Reports folder.

Key Features
Core Mechanics: Fluid character movement, jumping, and collision detection.

Combat System: Sprite-based weapon logic and ability execution.

Enemy AI: Patrol patterns and player detection.

Persistent HUD: Real-time tracking of health, score, and level progress.
